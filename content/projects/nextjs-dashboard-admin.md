## Overview

A modern and responsive admin dashboard built with the latest Next.js 15, showcasing best practices for authentication, data visualization, and responsive design patterns.

## Key Features

- **JWT Authentication**: Secure authentication system with JSON Web Tokens
- **Responsive Design**: Fully responsive interface that works seamlessly across all devices
- **Data Visualization**: Interactive charts and graphs for data insights
- **Next.js 15**: Built with the latest Next.js features including App Router
- **TypeScript**: Fully typed codebase for better development experience
- **Modern UI**: Clean and intuitive user interface

## Tech Stack

- Next.js 15
- TypeScript
- JWT Authentication
- React
- Tailwind CSS (assumed)
