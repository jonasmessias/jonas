Built a complete NFT minting platform with the following features:

- Direct smart contract interaction using ethers.js
- Multi-wallet support (MetaMask, Coinbase, WalletConnect)
- Real-time minting status and transaction tracking
- PFP (Profile Picture) generator tool
- Animated landing page with GSAP
- Responsive design with Tailwind CSS

The platform handled over 10,000 mints in the first 24 hours with zero downtime.
