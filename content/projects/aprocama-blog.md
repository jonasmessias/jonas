## Overview

Website institucional completo desenvolvido para uma Associação de Produtores Rurais (Aprocama). A plataforma oferece um sistema de blog com gestão de conteúdo em Markdown, área de transparência, informações sobre serviços e formulário de contato integrado, tudo com animações suaves e design responsivo.

## Key Features

- **Sistema de Blog**: Gestão de posts com Markdown, Gray Matter e renderização HTML com Remark
- **Animações Modernas**: Transições e animações fluidas com Framer Motion
- **Formulário de Contato**: Integração com EmailJS para envio de mensagens
- **Validação de Dados**: React Hook Form com Zod para validação robusta de formulários
- **Componentes Reutilizáveis**: UI components com Radix UI (Dialog, Dropdown Menu, Navigation Menu)
- **Design System**: Tailwind CSS 4 com Typography plugin e variantes de estilo
- **Área de Transparência**: Seção dedicada para prestação de contas e documentos
- **Performance Otimizada**: Next.js 15 com Turbopack para builds ultra-rápidos
- **SEO Friendly**: Estrutura otimizada para mecanismos de busca

## Tech Stack

- **Framework**: Next.js 15.5.2 (App Router)
- **Linguagem**: TypeScript 5
- **Styling**: Tailwind CSS 4 + @tailwindcss/typography
- **Animações**: Framer Motion 12
- **Formulários**: React Hook Form + Zod validation
- **UI Components**: Radix UI (@headlessui/react)
- **Markdown**: Gray Matter + Remark + Remark-HTML
- **Email**: EmailJS Browser
- **Icons**: Lucide React
- **Carrossel**: Swiper 11
- **Utilities**: clsx, tailwind-merge, date-fns
- **Deploy**: Vercel
